# Clipbrd

Welcome to the Clipbrd repository!

## Description

Clipbrd is a utility that connects your clipboard with ChatGPT and serves to provide a radically simple way of obtaining an answer without much work and being quite discreet.

## How to use

You must create a virtualenv and install the requirements provided, then copy example.env into .env and use your OpenAI API Key, then you can just build it into a binary and it should work for MacOS, Linux or Windows.
